CSS


CSS stands for Cascading Style Sheets, it is a file ending with .css that describes how HTML elements are to be displayed. It simplifies the personalization of HTML, with CSS the developer can design properties to HTML elements, from altering background color, font sizes to the position of elements on screen. It is mandatory during the development of a webpage. During this assignment CSS was used to design the colors of the navigation bar, the alignment of the headings and paragraphs, and the properties of the images and buttons (Dark Mode button).
